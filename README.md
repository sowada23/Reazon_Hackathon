# Salaryman Escape - Pixel Art Edition

会社員が定時退社を目指してオフィスを駆け抜ける、ブラウザ向けの2Dピクセルアートゲームです。

通常ステージでは、未読メール・山積み書類・締め切り時計などの障害物を避けながら書類を集めます。一定スコアに到達するとボス戦に移行し、部長、ITコンサルタント、取締役、クライアント、深夜残業の悪夢という全5フロアの相手を撃破して脱出を目指します。

## 主な機能 (Features)

- HTML Canvas を使った2D横スクロール風アクションゲーム
- 会社・オフィスをテーマにしたピクセルアート表現
- 通常ステージとボス戦で異なるゲームプレイ
  - 通常ステージ: スペースキーでジャンプし、障害物を回避
  - ボス戦: WASD または矢印キーで自由移動し、自動ショットで攻撃
- 全5段階のボスステージ
  - 1F: 怒りの部長
  - 2F: ポンコツなITコンサルタント
  - 3F: ムチャ振りな取締役
  - 4F: 鬼のクライアント
  - 5F: 深夜残業の悪夢
- 書類アイテムの収集システム
  - 書類5枚で「承認ハンコ弾」を発射
  - 書類5枚で「有給バリア」を発動
- コーヒーカップによるHP回復アイテム
- HP、スコア、書類枚数、ボスHPバーなどのゲームUI
- Web Audio API による8bit風BGM・効果音
- パーティクル演出、警告画面、ポーズ画面、ゲームオーバー画面、クリア画面
- `requestAnimationFrame` と固定更新レートによるゲームループ制御

## セットアップとインストール方法 (Getting Started)

### 前提条件

- モダンブラウザ
  - Google Chrome
  - Microsoft Edge
  - Firefox
  - Safari など
- Git
- 任意: ローカルHTTPサーバーを起動できる環境
  - Python 3 など

### 1. リポジトリをクローン

```bash
git clone <repository-url>
cd ReazonHackathon
```

### 2. 依存関係のインストール

このプロジェクトには `package.json` がなく、外部npmパッケージも使用していません。

そのため、依存関係のインストールは不要です。

### 3. 環境変数の設定

環境変数は使用していません。

`.env` ファイルの作成やAPIキーの設定は不要です。


画像ファイル名や配置場所を変更する場合は、`src/work_div.js` 内の画像読み込み部分も合わせて変更してください。

## 使い方・実行方法 (Usage)

### 方法1: HTMLファイルを直接開く

`index.html` をブラウザで開くとゲームを起動できます。

```bash
open index.html
```

macOS以外の環境では、ファイルマネージャーから `index.html` をダブルクリックして開いてください。

### 方法2: 簡易HTTPサーバーで起動する

画像アセットの読み込みをより安定させたい場合は、プロジェクトルートで簡易HTTPサーバーを起動してください。

```bash
python3 -m http.server 8000
```

ブラウザで以下にアクセスします。

```text
http://localhost:8000/
```

### 操作方法

| 状態 | 操作 | 内容 |
| --- | --- | --- |
| タイトル画面 | `Space` | ゲーム開始 |
| 通常ステージ | `Space` | ジャンプ |
| ボス戦 | `W` / `A` / `S` / `D` | プレイヤー移動 |
| ボス戦 | 矢印キー | プレイヤー移動 |
| ボス戦 | `Q` | 書類5枚を消費して承認ハンコ弾を発射 |
| 通常ステージ・ボス戦 | `E` | 書類5枚を消費して有給バリアを発動 |
| プレイ中 | `Esc` | ポーズ |
| ポーズ中 | `Space` | ゲーム再開 |
| ゲームオーバー・クリア画面 | `Space` | タイトル画面へ戻る |

### ゲームの流れ

1. タイトル画面で `Space` を押して開始します。
2. 通常ステージでは、障害物を避けながら書類を集めます。
3. 一定スコアに達すると警告画面が表示され、ボス戦に移行します。
4. ボス戦では自動ショットでボスを攻撃しながら、敵弾を回避します。
5. 書類を5枚集めるごとに、`Q` の承認ハンコ弾や `E` の有給バリアを使えます。
6. 全5体のボスを倒すとクリアです。

### 本番ビルド

ビルド工程はありません。

静的ファイル一式をそのままWebサーバーに配置することで公開できます。

```text

asset/
```
